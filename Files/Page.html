Basic Syntax

This is basic syntax for HTML

HTML Tag Description Example Usage
<!DOCTYPE html> Declares the document type and HTML version (HTML5). <!DOCTYPE html>
<html> Root element of an HTML page. <html> ... </html>
<head> Contains meta-information about the document (title, styles, scripts). <head> ... </head>
<title> Sets the title of the webpage (shown in the browser tab). <title>Document Title</title>
<body> Contains the visible content of the webpage. <body> ... </body>
<h1> to <h6> Defines different levels of headings. <h1>Heading 1</h1>
<p> Defines a paragraph of text. <p>This is a paragraph.</p>
<a> Defines a hyperlink. <a href="https://www.example.com">Link Text</a>
<img> Defines an image. <img src="image.jpg" alt="Description">
<table> Defines a table. <table> ... </table>
<tr> Defines a table row. <tr> ... </tr>
<th> Defines a table header cell. <th>Header Text</th>
<td> Defines a table data cell. <td>Data Text</td>

Example of a Basic HTML Structure:


                        <!DOCTYPE html>
                        <html lang="en">
                        <head>
                            <meta charset="UTF-8">
                            <title>Interactive HTML Guide</title>
                            <meta name="description" content="THis is Project Learning HTML">
                            <meta name="keywords" content="HTML">
                            <meta name="author" content="Mostafa Hatem">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        </head>
                        <body>
                            <h1>Welcome to the Interactive HTML Guide</h1>
                            <p>This is a basic HTML structure example.</p>
                            <a href="https://www.example.com">Example Link</a>
                        </body>
                        </html>
